# Titanic Survival Prediction

## Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

## Dataset
Titanic Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Steps Performed
1. Loaded Titanic dataset
2. Handled missing values
3. Visualized survival distribution
4. Encoded categorical features
5. Trained Random Forest Classifier
6. Evaluated model performance

## Result
Model Accuracy: 79.88%
