# Sales Prediction Using Python

## Objective
Build a machine learning model to predict product sales based on advertising expenditure.

## Dataset
Advertising Dataset

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Steps Performed
1. Loaded dataset
2. Explored data
3. Visualized TV advertising vs sales
4. Trained Linear Regression model
5. Evaluated model performance
6. Predicted future sales

## Result
Successfully predicted sales using advertising data.
